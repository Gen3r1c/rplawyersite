# Salvation of Souls MC Website

This repository contains the Salvation of Souls MC website, built for GTA RP. The site showcases the club, its chapters, leadership, events, and a gallery of club activity.

## Purpose
The website was created to transform the old law office concept into a roleplay club site for Salvation of Souls MC. It is designed to support your MC's identity, display the patch clearly, and highlight the black and purple color theme.

## What’s Included
- Home page with the club brand, mission, and chapter overview
- About page with Salvation of Souls MC history and leadership story
- Gallery page with club event photos and a click-to-expand lightbox
- Events page for upcoming rides and meetings
- Contact page with chapter contact details
- Shared CSS styles for a consistent dark black and purple theme

## Why It Was Built
This site was rebuilt to match your GTA RP club vision:
- Replace the old law firm content with MC branding
- Use the patch as the central logo and identity element
- Make the site dark, bold, and easy to navigate
- Add a modern gallery experience for club photos

## Development
Open `index.html` in your browser or run a local web server from the project folder.
