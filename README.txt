Rayne Law Static Website

Quick start:
1) Open raynelaw_site/index.html in a browser.
2) Replace assets/img/hero.jpg with your banner image (keep filename).
3) In advisories.html, replace href="#" on the Open Advisory button with your Google Doc/PDF link.
4) Upload the folder to any static host (Netlify / Vercel / GitHub Pages / Cloudflare Pages).

Domain:
Point raynelaw.sa to your host following their DNS instructions.
